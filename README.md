# Welcome !

With this based **node.js** application, you can ask the console when you'll be in holiday anywhere around the world !


###   Installation

Open the console and put 

```
npm install @nicodehez/holidates
```



###  Run 

After the installation, you can write node . followed by the name of the country and the year, for exemple :


```
node . Belgium 2019
```
